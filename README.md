## 🌸 О проекте

**Сакура** — это конверсионный лендинг для онлайн-школы японского языка. Решение учить язык
рождается на эмоции — посмотрел аниме, спланировал поездку, увидел рекламу. Задача проекта:
за один экран превратить интерес в заявку и снять страх «японский — это сложно».

Каждая секция ведёт пользователя к одному целевому действию — **бесплатному первому уроку**.
Воздушный первый экран с параллакс-сценой (восходящее солнце, Фудзи, тории, фонарики, сакура)
задаёт настроение, а дальше — прозрачные форматы обучения, тарифы без скрытых платежей, живые
отзывы и интерактивный тест уровня прямо на странице.

## ✨ Особенности

- 🎌 **Японская эстетика** — палитра васи-бумаги, киновари, сакуры, индиго, матчи и золота
- 🌸 **Падающие лепестки сакуры** — лёгкая CSS-анимация поверх ключевых секций
- 🏔️ **Параллакс-сцена в Hero** — солнце, Фудзи, тории, фонарики и волны движутся с разной скоростью
- 🔢 **Анимированные счётчики** — статистика оживает при попадании в зону видимости
- 🎴 **Tilt-карточки курсов** с кандзи-штампами (個 話 試 心)
- 💬 **Карусель отзывов** — автопрокрутка, стрелки, точки, пауза по наведению
- 🧪 **Интерактивный тест уровня** — мини-вопрос с мгновенной проверкой ответа
- 📊 **Аккордеон FAQ** с плавным раскрытием
- 📝 **Форма заявки** с валидацией и graceful-фолбэком (работает даже без бэкенда)
- 🈷️ **Бегущая строка иероглифов** с переводом
- 📱 **Полная адаптивность** — mobile-first, без горизонтального скролла
- ♿ **Уважение к `prefers-reduced-motion`** — анимации отключаются для пользователей с такой настройкой

## 🚀 Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2a. Запустить только фронтенд (форма работает с graceful-фолбэком)
npm run dev

# 2b. Или фронтенд + бэкенд заявок одновременно
npm run dev:all
```

| Адрес | Что |
|-------|-----|
| http://localhost:5173 | Фронтенд (Vite) |
| http://localhost:3001 | Бэкенд заявок (Express) |

Заявки с формы сохраняются в `server/data/leads.jsonl`. Если бэкенд не запущен — форма всё
равно покажет успешную отправку (демо-режим), чтобы лендинг работал автономно.

### Сборка для продакшена

```bash
npm run build      # сборка в dist/
npm run preview    # локальный предпросмотр прод-сборки
```

## 🧩 Архитектура

Весь лендинг построен на **единой дизайн-системе**: общие компоненты и токены палитры/типографики
переиспользуются во всех секциях, поэтому ритм движения и визуальный язык одинаковы от шапки до футера.

```
.
├── index.html                  — точка входа, подключение шрифтов
├── tailwind.config.js          — палитра, шрифты, кастомные анимации
├── server/
│   └── index.js                — Express-бэкенд приёма заявок
└── src/
    ├── App.jsx                 — сборка секций
    ├── index.css               — Tailwind + дизайн-токены + keyframes
    ├── lib/
    │   └── variants.js         — общие варианты анимаций Framer Motion
    └── components/
        ├── ui/                 — переиспользуемые компоненты
        │   ├── Reveal.jsx          · появление при скролле
        │   ├── Counter.jsx         · анимированный счётчик
        │   ├── Marquee.jsx         · бесконечная бегущая строка
        │   ├── SakuraPetals.jsx    · падающие лепестки
        │   ├── Button.jsx          · кнопки (primary/ink/outline/ghost)
        │   ├── Section.jsx         · обёртка секции
        │   ├── SectionTitle.jsx    · заголовок с кандзи-акцентом
        │   └── decor.jsx           · SVG-декор (тории, Фудзи, сакура, волны…)
        └── sections/           — 13 секций лендинга
            ├── Navbar.jsx          · Hero.jsx        · LangMarquee.jsx
            ├── About.jsx           · Stats.jsx       · Courses.jsx
            ├── Pricing.jsx         · LevelTest.jsx   · Testimonials.jsx
            ├── FAQ.jsx             · Consultation.jsx
            └── FinalCTA.jsx        · Footer.jsx
```

## 📋 Секции

1. **Navbar** — sticky, сжимается при скролле, анимированное мобильное меню
2. **Hero** — параллакс-сцена, плавающие бейджи «бесплатно» и «JLPT N5–N1»
3. **Бегущая строка** — встречные ленты иероглифов с переводом
4. **О школе** — основательница, методика, преимущества
5. **Статистика** — анимированные счётчики достижений
6. **Виды обучения** — 4 формата с кандзи-штампами
7. **Тарифы** — 3 плана, центральный выделен как «Хит продаж»
8. **Тест уровня** — интерактивный мини-вопрос
9. **Отзывы** — карусель с автопрокруткой
10. **FAQ** — аккордеон частых вопросов
11. **Форма заявки** — с валидацией и статусами отправки
12. **Финальный CTA** — драматичный призыв с тории и солнцем
13. **Footer** — контакты, меню, гигантский фоновый кандзи
